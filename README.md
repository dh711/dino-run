# Dinosaur Jump!

Chrome's Dinosaur Game ported to Arduino.

## Resources.
1. Arduino board.
2. 16x2 LCD Display.
3. Push button.
4. Some wires.

## Features.
* Game speeds up as the score increases.
* Randomly spawns cacti or crows.
* No cheating! Button cannot be held to be in air forever.
